# SysFloodv

SysFloodv is a Python script that allows you to conduct a simple SYN flood attack using Scapy.

## Usage

1. Make sure you have Python 3 installed on your system.
2. Install Scapy by running `pip install scapy`.
3. Clone or download the `sysflood.py` script to your local machine.
4. Open a terminal and navigate to the directory containing `sysflood.py`.
5. Run the script using the command `sudo ./sysflood.py`. (Note: sudo privileges are required for sending raw packets.)
6. Follow the on-screen instructions to enter the victim's IP address.
7. The script will start flooding the victim's IP address with SYN packets.
